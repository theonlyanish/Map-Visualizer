About Starfall Atlas

Starfall Atlas is a live GitHub activity visualizer for open-source discovery. It maps public repository events into an interactive code atlas with region-based exploration.

Visitors can browse live GitHub events, switch between Starfall and Galaxy views, filter pushes, forks, pull requests, issues, comments, and releases, and open repository pages directly from the atlas.

The site uses GitHub's public Events API, a lightweight proxy, and local classification rules to group repositories into stable visual regions such as machine learning, frontend, infrastructure, security, data, mobile, devtools, and systems.
